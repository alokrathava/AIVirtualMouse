# AiVirtualMouse
AI/ML Virtual Mouse Using Python

Dependencies required <br>
# 1. opencv 
  ```
  pip install opencv-python
  ```
  
 # 2. numpy
   ```
   pip install numpy
   ```
   
 # 3. autopy (Python 2.7, or Python 3.5 and up.)
   ```
   pip install autopy
   ```
   
# 4. mediapipe
     
     pip install mediapipe
